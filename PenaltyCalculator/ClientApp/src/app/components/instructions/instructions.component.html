The instructions for using this penalty calculating utility for libraries

  • This is a API-driven utility that calculates your penalty by the number of business days you borrowed the book
  • Please read the following instructions carefully to use this utility:
  • Enter the checkout date, when you got the book from the library in the first input field
  • Enter the return date, when you returned the book to the library in the second input field
  • Select your region by choosing a country from the dropdown menu
  • Once you have entered the correct required information, click the "Calculate Penalty" button to get the penalty amount
